# LTAK
Linux-based Team Awareness Kit

TAK build specifically for Linux-based systems. Distro-specific packages coming soon for:

- Fedora
- Arch
- Gentoo
- Debian
- Slackware

Features to be completed:

- Native Meshtastic support.
- Native APRS support.
- Real-time markers and polygon drawing.
- ESRI integration.
- Plugin support.
- P2P support for serverless functionality.
- Radio usage with direct support for the Yaesu DR-2X with LAN unit, or similar repeater hardware.
